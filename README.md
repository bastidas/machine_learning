# machine_learning

Simple machine learning tools demonstrated on the classic iris data set.

The classifiers used:
	- Naive Bayes
	- Nearest Neighbors
	- RBF SVM
	- Linear SVM
	- Decision Tree
	-  Random Forest
	- AdaBoost
	- Neural Net

Files in this repo:
	- ml_iris.py
	- ml_plot_tools.py
	- ml_classifier_methods.py
